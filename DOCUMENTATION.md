## ١. رؤية المشروع وهدفه

**وعي** منصة تعليمية إسلامية مجانية بالكامل، هدفها تقديم تعليم ديني منظم وموثوق للشباب المسلم.

### الشعار
> **"بما لا يسع المسلم جهله"**  
> تعلّم أساسيات دينك بطريقة منظمة وتفاعلية — مجاناً، وفي أي وقت.

### المبادئ الأساسية
- **مجانية كاملة** — لا اشتراكات، لا محتوى مقيد
- **منهجية** — مسار تعليمي متسلسل، ليس مجرد مقاطع عشوائية
- **تفاعلية** — اختبارات، ملاحظات، متابعة تقدم
- **موثوقة** — محتوى منتقى ومرتب بعناية من قِبَل فريق إدارة المنصة
- **عربية** — واجهة وتوجه عربي كامل مع دعم لغات مستقبلاً

## ٢. الجمهور المستهدف

### المستخدم الأساسي
**الشاب المسلم** الذي يريد:
- تعلم أمور دينه بشكل منهجي
- التفقه في الدين بعيداً عن الفوضى المعلوماتية
- شهادة تثبت إتمامه للمسار التعليمي

### خصائص المستخدم
- يتعامل مع الإنترنت والهواتف بشكل طبيعي
- يفضل المحتوى العربي
- مشغول — يحتاج محتوى قصير ومركّز
- يحتاج تحفيزاً (شهادات، تقدم مرئي)

## ٦. الشاشات والصفحات

### ٦.١ صفحة الهبوط (Landing)
**الغرض:** تعريف الزائر بالمنصة قبل التسجيل

**المحتوى:**
- Hero: شعار المنصة + وصف مختصر + زر "ابدأ الآن"
- قسم المواد: شبكة المواد الست مع اسم المدرس وعدد المحاضرات
- قسم الميزات: 6 ميزات مصورة (فيديو تفاعلي، اختبارات، شهادات...)
- Footer

**لا تتطلب تسجيل دخول**

---

### ٦.٢ صفحة المصادقة (Auth)
**الغرض:** تسجيل الدخول أو إنشاء حساب

**المسارات:**
- **تسجيل دخول:** بريد إلكتروني + كلمة مرور
- **إنشاء حساب:** الاسم + بريد إلكتروني + كلمة مرور + تأكيد كلمة المرور

**القرارات المهمة:**
- ✅ **لا توجد موافقة يدوية** — الطالب يدخل مباشرة بعد التسجيل
- ✅ الحساب الجديد يُنشأ بدور `student` تلقائياً
- ✅ رسائل الخطأ بالعربية

---

### ٦.٣ لوحة الطالب (Student Dashboard)
**الغرض:** نقطة التحكم الرئيسية للطالب

**التبويبات:**

#### 📚 المواد
- شبكة المواد الست
- لكل مادة: دائرة تقدم، اسم المدرس، عدد المحاضرات، حالة الاكتمال
- المادة التالية مقفلة حتى اكتمال السابقة
- زر "استأنف" للمادة الجارية

#### 🔔 الإشعارات
- قائمة الإشعارات من الإدارة
- مصفاة حسب الدور (للجميع / للطلاب فقط)
- تمييز المقروء / غير المقروء

#### 📡 اللقاءات المباشرة
- قائمة الأحداث القادمة
- لكل حدث: العنوان، المدرس، التاريخ، الوقت، زر "انضم"
- **زر الانضمام يفتح الرابط مباشرة** (Zoom / Meet / أي رابط)

#### 📊 نتائجي
- سجل كل الاختبارات المحلولة
- لكل نتيجة: المادة، الدرجة، تاريخ الحل، شارة نجاح/رسوب

#### 🏅 الشهادة
- إذا استوفى الشروط: عرض الشهادة + زر تحميل PDF
- إذا لم يستوفِ: قائمة تدريجية بالمتطلبات المتبقية

---

### ٦.٤ عرض المادة (Course View)
**الغرض:** مشاهدة المحاضرات وتتبع التقدم وحل الاختبارات

**المكونات:**

#### مشغل الفيديو
- YouTube IFrame مضمّن
- تتبع نسبة المشاهدة (80% = محاضرة مكتملة)
- الانتقال التلقائي للمحاضرة التالية (اختياري)

#### قائمة المحاضرات (Sidebar)
- كل المحاضرات مع شريط تقدم فردي
- علامة ✓ للمحاضرات المكتملة
- عدد الملاحظات على كل محاضرة
- المحاضرة الحالية محددة

#### لوحة الملاحظات
- إضافة ملاحظة عند الوقت الحالي للفيديو
- تعديل / حذف الملاحظة
- النقر على الوقت يعود للحظة تلك في الفيديو
- **الملاحظات محفوظة في Firestore** (مرتبطة بالمستخدم والمادة والمحاضرة)
- إشعار تلقائي عند وصول الفيديو لوقت ملاحظة محفوظة

#### زر الاختبار
- يظهر فقط عند إكمال 100% من محاضرات المادة
- يفتح QuizModal

### ٦.٥ نموذج الاختبار (Quiz Modal)
**ثلاث مراحل:**

**١. مرحلة التمهيد**
- عنوان الاختبار، المادة، عدد الأسئلة، الوقت، درجة النجاح

**٢. مرحلة الحل**
- مؤقت ثابت (يتحول للأحمر < دقيقة)
- شريط تقدم الأسئلة المجابة
- أسئلة اختيار متعدد (4 خيارات)
- تسليم تلقائي عند انتهاء الوقت

**٣. مرحلة النتائج**
- الدرجة الكلية + رمز نجاح/رسوب
- تفصيل كل سؤال: إجابتك ✓/✗ + الإجابة الصحيحة

---

### ٦.٦ لوحة الجوكر (Super Admin / Joker Dashboard)
**الغرض:** إدارة كاملة لكل شيء في المنصة

**التبويبات:**

#### 📊 نظرة عامة
- إحصائيات عامة: الطلاب المسجلون، الاختبارات المحلولة، الشهادات الصادرة
- آخر نشاط الطلاب
- أكثر المواد مشاهدة
- معدل اكتمال الكورسات
- أوقات الذروة (متى يتصل الطلاب)

#### 👥 الطلاب
- جدول كل الطلاب: الاسم، البريد، تاريخ التسجيل، نسبة التقدم العامة
- فلترة حسب: الاسم، نسبة التقدم، تاريخ التسجيل
- النقر على طالب: يفتح تفاصيل تقدمه في كل مادة
- حذف حساب طالب

#### 📚 المواد والمسارات
- قائمة كل المسارات والمواد الموجودة
- **إضافة مسار جديد** (Learning Path): اسم، وصف، ترتيب المواد
- **إضافة مادة جديدة**: العنوان، المدرس، الوصف، لون الهوية، رابط قائمة YouTube
- **إضافة محاضرات**: كل محاضرة لها عنوان + رقم في القائمة (playlist index)
- تعديل وحذف أي مادة أو محاضرة
- إعادة ترتيب المواد داخل المسار

#### 📝 الاختبارات
- إنشاء اختبار جديد (مرتبط بمادة)
- QuizBuilder: إضافة أسئلة + خيارات + الإجابة الصحيحة
- تعديل وحذف الاختبارات

#### 🔔 الإشعارات
- كتابة إشعار: عنوان + نص + نوع (معلومة / نجاح / تحذير / إعلان)
- الجمهور المستهدف: الكل / الطلاب / دور معين
- عرض عدد القرّاء لكل إشعار

#### 📡 اللقاءات المباشرة
- إضافة حدث: العنوان، المدرس، التاريخ، الوقت، رابط الانضمام، وصف، صورة
- تعديل وحذف الأحداث

#### 🏅 الشهادات
- الطلاب المستحقون للشهادة (أكملوا كل المواد والاختبارات)
- إصدار شهادة بنقرة واحدة
- عرض كل الشهادات الصادرة

#### 👤 الأدوار والصلاحيات
- قائمة كل المستخدمين مع أدوارهم
- تغيير دور مستخدم (ترقية / تخفيض)
- إضافة مدير جديد بدور محدد
- لا يمكن حذف أو تغيير دور الجوكر إلا بواسطة جوكر آخر

## ٨. منطق الأعمال الأساسي

### ٨.١ حساب تقدم المستخدم

```
نسبة المادة = (المحاضرات المكتملة / إجمالي المحاضرات) × 100
محاضرة مكتملة = نسبة المشاهدة >= 80%

نسبة الإجمالي = (مجموع المحاضرات المكتملة في كل المواد / مجموع كل المحاضرات) × 100
```

### ٨.٢ قفل المواد (Sequential Lock)
```
المادة [0] = متاحة دائماً
المادة [N] = متاحة فقط إذا اكتملت المادة [N-1] بنسبة 100%
```

### ٨.٣ شروط الشهادة
```
للحصول على الشهادة يجب:
✅ إكمال جميع مواد المسار بنسبة 100%
✅ اجتياز اختبار كل مادة (النجاح >= درجة النجاح المحددة)
```

### ٨.٤ تتبع مشاهدة الفيديو
```
كل ثانية: نحسب الموقع الحالي كنسبة من مدة الفيديو
عند الإيقاف أو تغيير المحاضرة: نحفظ النسبة في Firestore
الحد الفعّال = min(نسبة الموقع, نسبة الوقت المتراكم, 100%)
```
*هذا يمنع الغش بالتخطي للنهاية*

### ٨.٥ نظام الملاحظات الزمنية
```
عند الإضافة: نحفظ الثانية الحالية + النص في Firestore
```
